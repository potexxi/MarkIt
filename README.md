# MarkIt

**"MarkIt"** is a lightweight and user-friendly Markdown editor designed for writing and formatting documents efficiently.

Whether you're taking notes, writing documentation, or creating README files, MarkIt provides a clean editing experience with instant Markdown support.

## Features

* Supports text documents, especially Markdown files
* Personalize the editor with custom colors and themes
* Clean and intuitive interface
* Markdown syntax-buttons
* Easy editing of documents
* Adjust the performance-usage of animations
* Export your Markdown file to PDF

## Cloud-Sync

MarkIt includes an experimental cloud synchronization system with user accounts and cloud-storage. This feature was developed primarily as a learning project and therefore depends on a privately hosted server.

Please note that the server may not always be online or available. As a result, account creation, login, and cloud synchronization may be temporarily unavailable. Local editing and file management features are not affected.
